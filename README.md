# 📝 To-Do List App

A simple and clean To-Do List web application built with **HTML, CSS, and JavaScript**.

This app allows users to:

- ➕ Add tasks  
- ✏️ Edit tasks  
- 🗑️ Delete tasks  
- ✅ Mark tasks as completed  
- 💾 Save tasks in browser local storage  

---

## 🚀 Features

- Simple user interface
- Add task with button or Enter key
- Click task text to toggle complete/incomplete
- Edit existing tasks
- Delete tasks instantly
- Data persists after refresh using `localStorage`

---

## 🧱 Tech Stack

- **HTML** (`todo.html`)
- **CSS** (`todo.css`)
- **JavaScript** (`todo.js`)
- **Local Storage** (browser)

## ▶️ How to Run

1. Download or clone this project
2. Open the project folder
3. Double-click `todo.html`  
   **or**
4. Open with VS Code + Live Server (recommended)

---

## 💡 How to Use

- Type a task in the input field
- Click **Add** (or press **Enter**)
- Click task text to mark as completed
- Click **Edit** to modify a task
- Click **Delete** to remove a task

---

## 📌 Note

Tasks are stored in your browser's local storage.  
If you clear browser data/local storage, tasks will be removed.
